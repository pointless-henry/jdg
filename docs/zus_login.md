# Вход

В [eZUS][1] можно войти одним из доступных для иностранцев способов:

- логин и пароль
- профиль зауфаны
- через банк

Если какой-то способ временно не работает, то попробуйте другой.

!!! bug "Никак не получается войти в eZUS?"
    См. [возможные причины и решения][2].
